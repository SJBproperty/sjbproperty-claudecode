# Instagram Highlight Covers — Design Brief

## General Specs
- Dimensions: 1080 x 1920 px (story size, but only centre circle visible as cover)
- Safe zone: Centre 600 x 600 px circle
- Background: SJB Navy #0F222D
- Icon/text: White #FFFFFF
- Accent: Steel Blue #8CA6BB (optional ring or underline)
- Font: Montserrat Bold

## Cover 1: Services
- Icon: House with checkmark (or wrench icon)
- Text below icon: "SERVICES"
- For: Stories about what SJB offers (management, guaranteed rent, tenant find)

## Cover 2: Portfolio
- Icon: Building/property stack icon
- Text below icon: "PORTFOLIO"
- For: Property photos, before/after, refurb projects

## Cover 3: Track Record
- Icon: Bar chart trending up
- Text below icon: "RESULTS"
- For: Stats, milestones, achievements

## Cover 4: FAQ
- Icon: Question mark in circle
- Text below icon: "FAQ"
- For: Common landlord questions answered

## Design Notes
- Keep icons simple and recognisable at small size (24px circle on profile)
- All 4 covers must be visually consistent (same layout, same icon style)
- Minimalist style matching SJB brand — no busy patterns or gradients
- These will be created via Canva using brand colours
- Icon style: thin line / outline icons (not filled/solid) for a clean, modern look
- Text should be centred below the icon within the safe zone
- Consider adding a subtle Steel Blue #8CA6BB circular border around the icon for visual interest
