## What was built

Scraped rental listings from all 3 platforms (OpenRent, Rightmove, Zoopla) using Apify MCP actors.

### OpenRent (via Apify: vivid-softwares/openrent-property-scraper)
- 115 listings scraped from Stockport area, 110 imported after postcode filtering
- Landlord names captured for self-managing landlord identification

### Rightmove (via Apify: dhrumil/rightmove-scraper)
- Used dhrumil/rightmove-postcode-to-listing-url to resolve location identifiers
- Stockport (REGION 1268), M14 (OUTCODE 1568), M19 (1573), M20 (1575), M21 (1576), M22 (1577)
- 1,239 total listings scraped, 800 imported after postcode filtering
- Listing quality scores calculated (photos, description, floorplan, EPC)

### Zoopla (via Apify: getdataforme/zoopla-search-location-rent-scraper)
- 200 Stockport rental listings scraped and imported
- All 200 matched target postcodes

## Deviations

1. First Rightmove attempt (memo23/rightmove-scraper) returned 0 results - URL format incompatible. Used dhrumil actor ecosystem instead.
2. First Zoopla attempts (shahidirfan + powerai + hello.datawizards) all failed. getdataforme rent-specific actor succeeded.
3. Fixed extractPostcode to handle outcode-only patterns in addresses (e.g., "Stockport SK3" without incode).
