## What was built

Imported the Land Registry CCOD (UK Companies that Own Property in England and Wales) bulk dataset into the database, filtered to target postcodes.

### CCOD Import
- Source: `CCOD_FULL_2026_03.csv` (1.55 GB, 4,381,243 records)
- Streamed and filtered to SK1-SK8, M14, M19-M22 postcodes
- 27,319 property titles imported (corporate-owned in target area)
- 7,133 unique corporate landlord records created
- Company registration numbers preserved for Companies House cross-referencing

### Process
- Automated download failed (gov.uk requires account + licence acceptance)
- Sam downloaded manually from HM Land Registry service
- Streaming import processed 4.38M rows in ~5 minutes
- Added `relax_column_count` to CSV parser to handle trailing summary row

## Deviations

1. **Manual download required** — Gov.uk Land Registry service requires authenticated session and licence acceptance. All 3 automated methods failed as expected by the plan.

2. **CSV trailing summary row** — The CCOD CSV ends with "Row Count: 4381243" which has only 2 columns instead of 35. Added `relax_column_count: true` to gracefully skip malformed rows.
