Download the Land Registry CCOD (UK Companies that Own Property) bulk dataset and import corporate-owned property titles for target postcodes into the database.

Purpose: CCOD provides the most authoritative list of properties owned by companies (Ltd/LLP) in the target area — a key data source for identifying landlord companies.
Output: CCOD records in landlords and properties tables filtered to target postcodes.
</objective>
