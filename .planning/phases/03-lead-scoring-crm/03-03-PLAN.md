  <action>
    **Create scripts/export-hubspot.js** with exported function `exportHubspot(db, exportsDir)`.

    1. Query landlords with tired_score >= 50 (primary/ungrouped only):
    ```sql
    SELECT l.*,
      COUNT(DISTINCT p.id) as property_count,
      GROUP_CONCAT(DISTINCT p.current_energy_rating) as epc_ratings,
      MAX(p.void_days) as max_void_days
    FROM landlords l
    LEFT JOIN properties p ON p.landlord_id = l.id
    WHERE l.tired_score >= 50
      AND (l.match_group_id IS NULL OR l.is_primary_record = 1)
    GROUP BY l.id
    ORDER BY l.tired_score DESC
    ```

    2. For each landlord, build a HubSpot row:
    - **Name splitting:** If entity_type is 'ltd' or 'llp' and director_names exists, split first director name (format "SURNAME, Firstname" from CH) into First Name / Last Name. If individual, split landlord name on first space. Fallback: Company Name = landlord name, First/Last Name empty.
    - **follow_up_date:** Calculate as export date + 3 days using `new Date(Date.now() + 3*86400000).toISOString().split('T')[0]`
    - **lead_type:** Derive from btl_suitable and r2r_suitable flags
    - **pipeline_stage:** Always "New Lead"
    - **btl_suitable / r2r_suitable:** Convert 1/0 to "Yes"/"No"
    - **epc_rating:** Use worst EPC from aggregated epc_ratings (pick the letter closest to G)
    - **void_days:** Use max_void_days from query
    - **data_sources:** Use landlord.source field directly

    3. Write CSV using `csv-stringify/sync` (same pattern as build-lead-list.js) to `{exportsDir}/hubspot-import-{dateStamp}.csv`

    4. Also write a separate deals CSV `{exportsDir}/hubspot-deals-{dateStamp}.csv` with columns: Deal Name, Pipeline, Deal Stage, Contact Email, Amount. Deal Name = "PM - {landlord name}". Pipeline = "Property Management". Deal Stage = "New Lead". Amount = empty (not known yet).

    5. Log summary: total exported, BTL count, R2R count, both count, with email count, without email count (direct mail candidates).

    Export `exportHubspot(db, exportsDir)` function. Run directly when `require.main === module`.

    - Test lead_type derivation: BTL-only, R2R-only, both, neither
