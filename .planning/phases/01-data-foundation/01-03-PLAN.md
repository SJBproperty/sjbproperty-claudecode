Build the Companies House API integration that identifies property SPV landlords in target locations, classifies them as Ltd/LLP, fetches director details, and stores them in the landlords table.

Landlords table schema:
```sql
CREATE TABLE IF NOT EXISTS landlords (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  entity_type TEXT CHECK(entity_type IN ('ltd', 'llp', 'individual', 'unknown')) DEFAULT 'unknown',
  company_number TEXT,
  registered_address TEXT,
  sic_codes TEXT,
  source TEXT NOT NULL,
  source_ref TEXT,
  scraped_at TEXT NOT NULL,
  created_at TEXT DEFAULT (datetime('now')),
  UNIQUE(company_number)
);
```
