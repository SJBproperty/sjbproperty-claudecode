    Core function `exportStannp(db, exportsDir, batchSize = 30)`:

    1. SQL query joining landlords + properties:
       ```sql
       SELECT l.id, l.name, l.entity_type, l.owner_address, l.mailing_address,
         l.director_names, l.tired_score, l.btl_suitable,
         GROUP_CONCAT(DISTINCT p.address) as property_addresses,
         MIN(p.current_energy_rating) as worst_epc,
         COUNT(DISTINCT p.id) as property_count,
         MIN(p.address) as first_property_address,
         MIN(p.postcode) as first_postcode
       FROM landlords l
       LEFT JOIN properties p ON p.landlord_id = l.id
       WHERE (l.match_group_id IS NULL OR l.is_primary_record = 1)
         AND ${suppressionFilter()}
         AND l.btl_suitable = 1
         AND (l.owner_address IS NOT NULL OR l.mailing_address IS NOT NULL
              OR EXISTS (SELECT 1 FROM properties p2 WHERE p2.landlord_id = l.id))
       GROUP BY l.id
       ORDER BY l.tired_score DESC
       LIMIT ?
       ```

    2. `extractPostcode(address)` function: regex `/([A-Z]{1,2}\d[A-Z\d]?\s*\d[A-Z]{2})\s*$/i` to extract UK postcode from end of address string. Returns null if no match.

    3. `parseAddress(fullAddress)` function: extract postcode via `extractPostcode()`, remove postcode from string, split remaining on commas into line1, line2 (middle parts joined), city (last part). Return `{ line1, line2, city, postcode }`.

    4. `selectMailingAddress(lead)` function: waterfall -- if `lead.owner_address` exists use `parseAddress(lead.owner_address)`, else if `lead.mailing_address` exists use `parseAddress(lead.mailing_address)`, else use `parseAddress(lead.first_property_address + ', ' + lead.first_postcode)`.

    5. `formatName(lead)` function:
       - If entity_type is 'ltd' or 'llp' AND director_names is populated: split first director name (director_names may be comma-separated list -- take first) into firstname/lastname, set company to lead.name
       - If entity_type is 'ltd' or 'llp' WITHOUT director_names: set company to lead.name, firstname and lastname empty
       - If entity_type is 'individual' or 'unknown': split lead.name into firstname (first word) and lastname (remaining words), company empty

    6. Map each row to Stannp CSV columns:
       ```javascript
       {
         title: '',
         firstname: nameResult.firstname,
         lastname: nameResult.lastname,
         company: nameResult.company,
         address1: addr.line1,
         address2: addr.line2,
         city: addr.city,
         postcode: addr.postcode,
         country: 'GB',
         epc_rating: row.worst_epc || '',
         property_count: row.property_count || 0,
         property_address: row.first_property_address || '',
       }
       ```

    7. Write CSV using `csv-stringify/sync` with `{ header: true }` to `data/exports/stannp-btl-YYYY-MM-DD.csv`

    8. Console log summary: lead count, file path, address source breakdown (owner vs mailing vs property)

    9. Export function: `module.exports = { exportStannp, extractPostcode, parseAddress, selectMailingAddress, formatName }`
